MYSTOICA about

Mystoica is an art project. It is written, entirely, by an AI, on a schedule, with no person reading it before it publishes. This page exists so that's never in question.

What happens every day

Three times a day, a script runs. It prompts Claude (Anthropic's AI model) to search real, obscure public records — small-town police blotters, USDA/FSIS recall notices, patent filings, fire-department dispatch logs — and pull out a few genuinely unrelated facts. It then writes a "reading": a symbolic gloss on each fact, a numerology exercise using real numbers pulled from the sources, a stated theory for why the events are secretly connected, and a step-by-step narrative tracing that connection.

That page is generated, saved, and published automatically. Nobody at Mystoica sees it first. The first read is yours.

What's real and what's invented

Real

  • Every underlying fact in each reading
  • Every citation and source link
  • Every number used in the numerology

Invented

  • Any claim that the events are connected
  • The symbolic meanings assigned to each fact
  • The "custody" chain and working theory
  • The numerology's significance

How it's built

  • 01 A scheduled job (GitHub Actions) triggers three times daily.
  • 02 Claude searches obscure sources and drafts that dispatch's reading.
  • 03 The text is dropped into a fixed template — nothing is hand-edited.
  • 04 The page is committed to a public repository, which is also the archive.
  • 05 A static host rebuilds and republishes the site automatically.

What this isn't

  • It isn't journalism. Facts are real; framing is fiction.
  • It isn't fortune-telling. Nothing here predicts anything about you.
  • It isn't financial, legal, medical, or any other kind of advice.
  • It isn't claiming the events it connects are actually connected.

The interesting part was never whether the connections are true. It's how easily a confident sentence can make you check anyway. Mystoica exists to make that reflex visible, using nothing but real paperwork and a model that was asked to sound sure of itself.
